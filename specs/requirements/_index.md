# Requirements Index

Canonical specifications live under `specs/requirements/<domain>/`.

## Domains

- `CLI`
- `QA`
- `SYNC`
- `TUI`
- `WB`
- `WEB`

## Workflow

1. Author or update the specification in the appropriate domain folder.
2. Keep requirement clauses compact and trace links explicit.
3. Run `workbench validate` after changes.
